# 番茄专项研究

这个目录只解决一个问题：在番茄上写什么、怎么写、为什么这个方向更可能跑出来。

当前不直接进入正文。先把番茄平台机制、收益机制、AI 风险、题材榜单、读者承诺、开局留存和后续改编潜力拆清楚。

## 研究对象

1. 平台机制：推荐、完读、追更、互动、书名简介、标签和冷启动。
2. 收益机制：签约、分成、保底、完结奖励、听书/漫画/短剧等衍生收益。
3. AI 风险：平台规则、内容雷区、相似性风险、合同责任、审核风险。
4. 题材方向：都市高武、都市异能、异常事件、749/斩神相邻、悬疑智斗爽文。
5. 改编潜力：是否适合 AI 漫改、短剧改编、视觉化名场面、单集钩子。

## 研究顺序

1. 先复用文案蒸馏流程：证据抽取、source-open 通读归纳、route selector、runtime skill、clean-room 复刻、盲审回修。
2. 再用榜单样本拆“单书层”：这本书承诺什么、前 10 章怎么兑现、如何逼读者追更。
3. 再做横向对比：同赛道书的共同开局、主角能力、事件类型、对手结构、翻牌方式。
4. 然后归纳番茄可执行模型：新人能写什么，什么结构更稳，哪里容易自嗨。
5. 最后进入立项：题材、卖点、主角、能力体系、第一卷事件链、前 10 章钩子。

## 关键原则

不要把“题材热门”直接等同于“我们能写”。番茄立项要同时满足三件事：读者点得进、前 10 章留得住、后续能稳定连载。
