## 存放规则

1. 当前推进顺序、风险、暂停事项，放入 `00_总控/`。
2. 番茄平台机制、收益、AI 风险、赛道证据规格，放入 `10_番茄专项/`。
3. 小说蒸馏框架、runtime slot、叙述渲染器方法，放入 `20_小说蒸馏方法/`。
4. 早期讨论和旧规划，放入 `90_前期规划归档/`，不要继续追加新工作。
5. 样本池、榜单表、拆解表，放入 `../research/10_番茄专项/`。
