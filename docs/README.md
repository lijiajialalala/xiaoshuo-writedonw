## 存放规则

1. 当前推进顺序、风险、暂停事项，放入 `00_总控/`。
2. 番茄赛道证据规格和样本执行规则，放入 `10_番茄专项/`。
3. 小说蒸馏框架、runtime slot、叙述渲染器方法，放入 `20_小说蒸馏方法/`。
4. 三平台投稿、收益、推荐、合同和 AI 风险研究，放入 `30_平台研究/`。
5. 早期讨论和旧规划，放入 `90_前期规划归档/`，不要继续追加新工作。
6. 样本池、榜单表、拆解表，放入 `../research/10_番茄专项/`。
