# 三平台收益机制清单

## 0. 研究口径

这个文件只回答一个问题：

```text
作者在平台上到底有哪些钱可以拿？
```

每一项都按以下维度记录：

```text
收益来源
-> 触发条件
-> 到账/结算周期
-> 新人可达性
-> 更新成本
-> 断更/违规损失
-> 是否适合 AI 辅助高频生产
-> 可信度
```

金额和条件会变，正式投稿前必须重新核对官方福利页、后台、合同和编辑口径。

## 1. 番茄

### 1.1 稿费构成

可信度：high。

番茄官方把稿费拆成：

```text
番茄阅读收益
+ 番茄听书收益

+ 集团版权收益
+ 全勤收益
+ 买断/保底
+ 渠道收益
+ 版权收益
+ 其他收益
```

来源：[平台稿费&福利内容大揭秘](https://fanqienovel.com/writer/zone/article/7451451948726763545)。

### 1.2 阅读/听书收益

| 项 | 内容 |
| --- | --- |
| 收益来源 | 番茄小说APP阅读和听书产生的广告收益分成 |
| 触发条件 | 作品签约、上架、获得阅读/听书流量 |
| 结算周期 | 每个自然月第二个周四24点前发放上月稿费，节假日顺延 |
| 新人可达性 | 中等，先过签约和推荐验证 |
| 更新成本 | 高，需要稳定更新并维持读完率/追读率 |
| 断更/违规损失 | 影响推荐、福利和后续验证 |

| 类型 | 内容 |
| --- | --- |
| 集团渠道收益 | 今日头条、抖音、番茄畅听、番茄畅听音乐版、红果短剧、悟空浏览器等集团渠道产生的听读收益 |

生产影响：

1. 番茄不只是阅读平台，还连接听书、短剧、漫画、有声、海外等版权通道。
2. 我们确定小说方向时，短剧/漫画/有声潜力不能压过追读，但应作为加分项。
3. 异常事件、都市高武、规则破解路线如果视觉化强，理论上更适合后续版权开发。

### 1.5 全勤奖

可信度：high。

番茄2026年度福利说明写明，全勤奖、星火计划按现行标准继续核算和发放；2025年度福利页中的全勤和星火表格可作为当前执行口径，正式投稿前仍需后台复核。

2025官方能核到的全勤条件：

1. 奖励对象为番茄小说独家原创首发签约小说。
2. 签约作品有效更新字数满10万字的次月起才可获得全勤奖励。
3. 自然月内番茄小说App内听书收益与阅读分成收益之和需达到500元。
4. 自然月内每日有效更新字数需达到4000字或6000字对应档位。
5. 保底作品上一自然月为超保底状态，下一自然月才可按规则获得全勤。
6. 同一自然月每位作家仅一部签约作品享受全勤奖励。
7. 违规、停止推荐、下架、刷收益等情况会影响福利发放。

当前可核金额：

| 类型 | 更新要求 | 奖励金额 |

| 普通全勤奖 | 满足普通全勤条件的自然月，每日有效更新4000字及以上 | 600元 + 作品月听读收入 * 5%，上限1500元/月 |
| 进阶全勤奖 | 满足普通全勤条件的自然月，每日有效更新6000字及以上 | 800元 + 作品月听读收入 * 5%，上限2000元/月 |

### 1.7 AI 辅助创作边界

可信度：high。

番茄官方一方面上线了AI写作工具，明确其用途包括开书灵感、大纲、卡文锦囊、资料查询、AI起名、改写扩写、自定义描写、续写等，并说明作者最终是否使用工具生成内容、如何使用该内容由作者自行确定。

另一方面，番茄签约流程优化说明明确：平台会对情节套路化特征显著、叙事逻辑混乱、大篇幅使用AI生成的低质内容开展长期治理；自2025年4月7日起，在内容签约环节增加审核机制，对不符合平台要求的低质内容作出拒绝签约、内容下架等处罚，多次发布低质内容的作者将不再予以签约机会。

番茄平台内容发布规范进一步把“低质/水文”拆成可执行口径：批量发布重复或高度相似内容、连续或大量发布无意义/低质内容、内容拼凑且前后无衔接、复制粘贴百科/新闻/教材教程、使用非正常手段连续发布大量低质且逻辑不连贯内容、剧情节奏异常和流水账叙事、大量无关环境描写或空洞升华、人物单薄或前后主角不一致、经大量用户举报并被人工核实为严重影响阅读体验等，都可能进入处置范围。

来源：

1. [AI写作工具功能上线通知](https://fanqienovel.com/writer/zone/article/7327136545129906238)
2. [番茄签约流程优化说明](https://fanqienovel.com/writer/zone/article/7490454318521270296)
3. [番茄平台内容发布规范](https://fanqienovel.com/writer/zone/article/7237084151751376957)

生产结论：

1. AI辅助本身不是番茄官方禁止项。
2. 大篇幅AI低质内容、逻辑割裂、套路化、批量洗稿，是签约和账号信用风险。
3. 我们的AI流程必须保留人工主创判断、证据归纳、相似风险检查和正文质量门，不能把AI当成自动铺文机器。
4. 低质风险不是抽象的“AI味”，而是可以被拆成剧情衔接、人物一致性、信息密度、重复度、无关描写占比、章节有效推进等质量检查项。

### 1.8 抄袭、洗稿和违规内容处罚边界

可信度：high。

番茄抄袭处罚说明明确，抄袭不只包括照搬文字，也包括将具体文字打乱后同义替换、故事背景/人物设定/剧情走向存在内容相似、抄袭自己已发表旧稿、续写他人原创作品、搬运或改写影视/动漫/有声/短视频/互动游戏等其他形式作品内容。处罚可包括作品屏蔽下架、追回违规稿费、取消当月及此后创作福利，严重时封禁作者账号。

番茄低俗等违规内容处罚说明还提到，为了维持全勤而上传涉黄低俗、违反法律法规或政策内容、复制粘贴他人或网络平台设定/描写等，会带来停止推荐分发、取消福利奖金、作品下架、账号封禁永不合作等风险。

来源：

1. [作品涉嫌抄袭相关处罚说明](https://fanqienovel.com/writer/zone/article/7147522827501961223)
2. [作品发布低俗等违规内容行为处罚](https://fanqienovel.com/writer/zone/article/7218124710930808892)

生产结论：

1. clean-room 不是可选项，而是投稿安全底线。
2. 样本研究只能抽象出生成机制，不能迁移高识别角色设定、能力设定、异常规则、剧情顺序、翻牌路径和名场面。
3. 为全勤硬凑正文会同时触发低质、水文、内容安全和福利取消风险。

### 1.9 番茄收益结论

番茄的钱主要来自：

```text
广告听读分成
+ 福利激励
+ 新媒体拉新
+ 集团渠道
+ 版权/IP
```

对新人最现实的顺序：

```text
先过签约
-> 进推荐验证
-> 争取读完率/追读率
-> 拿到基础听读收益
-> 再谈全勤/星火/拉新/版权
```

## 2. 七猫

### 2.1 签约和收益模式

可信度：high。

七猫官方模式：

1. 保底+超保底分成。
2. 纯分成。
3. 超级纯分成。

保底签约作品千字价格官方口径为20元到1000元不等。七猫官方解释保底是预付款，作品成长期给基础保障，成熟期超保底后按分成共享收益。

来源：[七猫中文网&奇妙小说网2025年作家福利](https://www.qimao.com/welfare-police/)。

### 2.2 结算和发放

| 项 | 内容 |
| --- | --- |
| 稿费发放 | 按自然月统计，次月8日发放全部稿费 |
| 收款方式 | 发放至签约时提供的银行卡 |
| 保底作品连载中 | 累计分成超过累计保底后，当月分成多发分成，否则多发保底 |
| 保底作品完结后 | 先做收支平衡，累计分成超过总稿费收入后，超出部分按分成结算 |

### 2.3 版权和打赏

| 收益项 | 官方口径 |
| --- | --- |
| 全版权收益 | 精品/知名作者保底+超保底分成模式下，全版权50%收益分成 |
| 现金打赏 | 按现金价值与作者五五分成 |
| 苹果端打赏 | 扣除30%手续费后五五分成 |
| 纵横捧场道具 | 七猫/奇妙签约作品在纵横小说APP产生的捧场收益，作者分70% |

### 2.4 全勤奖

七猫全勤比番茄更像“按合同类型和数据指标分层”的收益系统。

| 类型 | 关键条件 | 奖励逻辑 |
| --- | --- | --- |
| 保底全勤奖 | 保底独家签约作品，无断更、灌水、抄袭等不良记录 | 日更4000-6000字为5元/千字，日更6000字以上为7元/千字 |
| 纯分成阶梯全勤奖 | 纯分成独家签约作品，日更4000字以上，均留/UV等达标 | 根据平均留存、日阅读UV、类型和阶段，6元/千字到200元/千字不等 |
| 超级纯分成全勤奖 | 超级纯分成独家签约作品，日更4000字以上，按自平台千字收入分档 | 从10元/千字到1000元/千字不等 |

限制：

1. 每月最高统计20万字全勤奖。
2. 每月允许1天请假。
3. 同一作者同一月连载作品超过一本，按全勤高的作品发放。
4. 七猫和奇妙两个站点仅可享受一本作品的全勤福利。
5. 纯分成、超级纯分成签约作品若完结总字数小于20万字或被判定烂尾，完结月不发全勤。
6. 有违规行为、异常数据的作品不发全勤。

### 2.5 月票奖和扶持

七猫官方列出：

1. 纵横小说APP月票奖，月票榜第1名现金15000元，第2名10000元，第3名8000元，第4-5名5000元，第6-7名3000元，第8-10名2000元，第11-20名1000元，并配套流量资源。
2. 分成转保底和保底提价机会。
3. 征文活动高额全勤，最高千字200元。
4. 扶摇计划、爆款作品扶持、版权衍生开发、个人品牌建设等更多扶持。

### 2.6 七猫收益结论

七猫的钱主要来自：

```text
保底/超保底
+ 分成收益
+ 阶梯全勤
+ 月票/捧场
+ 征文/扶持
+ 版权衍生
```

对新人最现实的路径：

```text
先准备2万字+大纲内投
-> 争取更好的签约模式
-> 用更新和留存跑全勤/分成
-> 再争取转保底、提价和版权
```

七猫对 AI 辅助的关键风险不是“能不能写得快”，而是：

1. 官方严禁工作室投稿。
2. 同一本书多人写作会触发严重风险。
3. 灌水、抄袭、异常数据、烂尾都会影响收益。

### 2.7 签约流程和首发风险补核

可信度：high。

七猫官方新人签约流程说明补充了几个直接影响投稿策略的点：

1. 签约审稿需要多轮评估，稿件多时会慢；超过7个工作日未收到回复，应主动联系投稿编辑。
2. 七猫一般收独家首发作品；作品如果在其他平台发布过但未签约，需要删除下架所有外站作品后再投稿，否则无法通过签约。
3. 未签约作品不会在七猫上架展示，避免作者在本站首发后无法签约而影响外站签约。
4. 合同审核一般5个工作日内完成。
5. 合同审批期间若被审核小组核查涉及抄袭，或已在外站发表并被盗版抓取全本，无法签约。

来源：[新人必看：七猫&奇妙签约流程详解](https://www.qimao.com/gonggao/63eb4a8b30bfb8280e3179dd/)。

这进一步说明：七猫不适合拿同一份稿件做随意多平台测试。若要投七猫，前期就应准备独家首发内投包。

### 2.8 七猫保底和题材口径补核

可信度：medium。

七猫主站2025福利页给出的保底范围为20元-1000元/千字；梧桐中文网收稿页给出的保底范围为20元-500元/千字，属于七猫旗下不同入口/站点口径，不能简单合并成一个“真实报价”。

七猫2025福利页男频编辑收稿口径中，明确有编辑主收都市高武、规则怪谈等方向，但也出现“规则怪谈文（过稿难）”的提醒。

生产结论：

1. 七猫不是不能投都市高武/异常事件，但规则怪谈/异常类可能更看重题材匹配、逻辑和编辑口味。
2. 保底报价不能从网上经验倒推，必须以内投编辑反馈和合同为准。
3. 对我们来说，七猫更适合作为“编辑商业判断”对照，而不是第一轮算法验证平台。

## 3. 起点/阅文

### 3.1 官方可核收益项

可信度：medium。

当前可公开核到的完整福利说明主要来自起点中文网2020年“网络作家星计划”。这是官方来源，但时间较旧，正式投稿前必须复核当前后台和合同。

来源：[阅文集团起点中文网作家星计划](https://acts.qidian.com/2020/2020_09/index.html)。

### 3.2 作品扶持

| 收益项 | 官方旧口径 | 生产影响 |
| --- | --- | --- |
| 作家扶持 | 标准分成VIP作品，作品入V后，在起点自有平台新增单章订阅和免费收入总稿酬的20%，次月发放 | 入V和稳定VIP更新是前提 |
| 创作补贴 | 签约新书入V后，前3个月每月1500元，最多4500元 | 新人入V后有短期保障 |
| 创作激励 | 新书入V后第4个月起，满足更新和均订/收入条件，每月1000元 | 需要付费读者基础，不是无门槛全勤 |
| 文以载道 | 现实题材、传统文化及其他个性创作内容，长篇5万元起保障合约 | 更偏特定题材和编辑评估 |

关键条件：

1. 每个自然月VIP作品每日有效VIP更新字数达到4000字以上。
2. 长篇作品不能连续两个自然月单月VIP有效更新总数少于6万字。
3. 创作激励要求均订大于等于500，或当月渠道收入加全部免费收入总和大于等于2000。

### 3.3 月票和道具

起点官方旧口径包括：

1. 月票奖。
2. 道具分成。
3. 互动激励。

月票奖旧表中，起点月票榜/新书月票榜前10名有2500元到10000元不等现金奖励，并随当月稿酬发放。

### 3.4 版权/IP

起点/阅文的核心优势不是基础全勤，而是付费读者、作者品牌和IP生态。

官方作家星计划列出的品牌运营和IP服务包括：

1. 曝光资源。
2. 运营基金。
3. 版权评估报告。
4. 专属版权经纪人。
5. 版权推广策划。
6. 下游IP定制合作机会。
7. 粉丝运营和IP出圈传播。

这对新人不一定立即可达，但决定长期天花板。

### 3.5 起点收益结论

起点的钱主要来自：

```text
VIP订阅
+ 作家扶持/补贴/激励
+ 月票和道具
+ 版权/IP
+ 作者品牌运营
```

对新人最现实的路径：

```text
先过签约
-> 建立收藏/追读
-> 入V
-> 维持每日VIP更新
-> 用订阅和均订证明价值
-> 再争取福利、月票和IP机会
```

公开信息不足的地方：

1. 当前最新全勤/补贴规则。
2. 当前起点男频/女频主动申请签约门槛。
3. 当前编辑内投偏好。
4. 当前AI辅助创作政策。
5. 当前合同对多平台发表和版权授权的实际约束。

这些必须列入后续验证清单。

### 3.6 当前公开资料边界

可信度：medium。

截至2026-05-10，本轮公开检索能确认：

1. 阅文作家专区当前仍是官方入口，并继续展示“加入网络作家星计划，真金白银助你成长”的福利入口。
2. 可公开核到的详细福利说明仍主要是2020年职业作家星计划，包括20%返奖、4500元创作补贴、1000元创作激励、文以载道计划、月票奖、道具分成等。
3. 本轮未找到比2020星计划更新、且同样完整公开的起点2026福利细则。

来源：

1. [成为作家](https://page.write.qq.com/intro)
2. [重磅 | 阅文集团作家福利体系全面升级](https://write.qq.com/portal/newscontent/18081035808294501)
3. [阅文集团起点中文网作家星计划](https://acts.qidian.com/2020/2020_09/index.html)

生产结论：

1. 起点不能用“2026最新规则已确认”来指导收益预期。
2. 只能把起点作为付费生态和长期追订对照样本。
3. 真要投起点前，必须通过作家助手后台、编辑口径或当前合同补核签约、上架、补贴、AI辅助和版权规则。

## 4. 三平台收益对比

| 维度 | 番茄 | 七猫 | 起点/阅文 |
| --- | --- | --- | --- |
| 主收入模型 | 免费阅读/听书广告分成 | 免费广告+保底/分成 | VIP订阅+付费生态 |
| 新人第一关 | 2万字签约机会 | 2万字+大纲投稿或直发申签 | 签约/收藏/编辑审稿 |
| 推荐/流量 | 8万字后推荐评估和7天验证 | 上架推荐+编辑/平台扶持 | 编辑、榜单、读者付费追读 |
| 稳定更新要求 | 4000/6000字、10万字后全勤门槛 | 日更4000起，月上限20万字全勤 | VIP日更4000、长篇月更约束 |
| 奖励重心 | 全勤、星火、拉新、短剧、完本续签 | 阶梯全勤、保底、月票、扶摇 | 创作补贴、创作激励、月票、IP |
| AI高频风险 | 高频发文、超长篇幅、恶意水文会影响活动资格 | 工作室、灌水、异常数据、烂尾风险高 | 付费读者和作者品牌对低质AI文容忍低 |
| 新人可达性 | 高，但竞争强且数据反馈硬 | 中，内投有编辑筛选和合同差异 | 中低到中，付费生态门槛更高 |

## 5. 对我们的直接结论

当前仍可以把番茄作为第一训练场，但理由要从“流量大”改成：

```text
番茄的直发、签约节点、推荐验证、读完率/追读率、全勤/星火/拉新/短剧激励，最适合我们先验证一个商业开局和生产运行包是否能跑。
```

但运行包不能被番茄绑死。七猫和起点分别提供两个反向校验：

1. 七猫校验“前2万字+大纲+编辑商业判断”。
2. 起点校验“付费读者追订+长期设定可信度+作者品牌”。
