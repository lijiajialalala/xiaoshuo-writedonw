# 三平台收益机制清单

## 0. 研究口径

这个文件只回答一个问题：

```text
作者在平台上到底有哪些钱可以拿？
```

每一项都按以下维度记录：

```text
收益来源
-> 触发条件
-> 到账/结算周期
-> 新人可达性
-> 更新成本
-> 断更/违规损失
-> 是否适合 AI 辅助高频生产
-> 可信度
```

金额和条件会变，正式投稿前必须重新核对官方福利页、后台、合同和编辑口径。

## 1. 番茄

### 1.1 稿费构成

可信度：high。

番茄官方把稿费拆成：

```text
番茄阅读收益
+ 番茄听书收益
+ 互动收益
+ 集团渠道收益
+ 集团版权收益
+ 全勤收益
+ 买断/保底
+ 渠道收益
+ 版权收益
+ 其他收益
```

来源：[平台稿费&福利内容大揭秘](https://fanqienovel.com/writer/zone/article/7451451948726763545)。

### 1.2 阅读/听书收益

| 项 | 内容 |
| --- | --- |
| 收益来源 | 番茄小说APP阅读和听书产生的广告收益分成 |
| 触发条件 | 作品签约、上架、获得阅读/听书流量 |
| 结算周期 | 每个自然月第二个周四24点前发放上月稿费，节假日顺延 |
| 新人可达性 | 中等，先过签约和推荐验证 |
| 更新成本 | 高，需要稳定更新并维持读完率/追读率 |
| 断更/违规损失 | 影响推荐、福利和后续验证 |
| AI 高频适配 | 只适合作为辅助，不适合无控制高频发文 |

### 1.3 互动收益

| 项 | 内容 |
| --- | --- |
| 收益来源 | 读者打赏礼物 |
| 触发条件 | 读者付费打赏 |
| 分成 | 去除运营商渠道费后，作者获得整体打赏收益的70% |
| 结算周期 | 和月稿费一起结算 |
| 新人可达性 | 低到中，依赖读者喜欢程度 |
| 风险 | 不应作为新人主收入假设 |

### 1.4 渠道和版权收益

| 类型 | 内容 |
| --- | --- |
| 集团渠道收益 | 今日头条、抖音、番茄畅听、番茄畅听音乐版、红果短剧、悟空浏览器等集团渠道产生的听读收益 |
| 集团版权收益 | 改编为红果短剧、真人有声、海外翻译等集团版权收益 |
| 外部渠道收益 | 作品在外部网站或平台分发后的收益 |
| 外部版权收益 | 真人有声、广播剧、游戏、动画、漫画、表情包动态漫、纸质出版、海外翻译、长剧、短剧、新媒体短剧等 |

生产影响：

1. 番茄不只是阅读平台，还连接听书、短剧、漫画、有声、海外等版权通道。
2. 我们确定小说方向时，短剧/漫画/有声潜力不能压过追读，但应作为加分项。
3. 异常事件、都市高武、规则破解路线如果视觉化强，理论上更适合后续版权开发。

### 1.5 全勤奖

可信度：high；具体金额档位待后台/福利图表复核。

2025官方文字能核到的条件：

1. 奖励对象为番茄小说独家原创首发签约小说。
2. 签约作品有效更新字数满10万字的次月起才可获得全勤奖励。
3. 自然月内番茄小说App内听书收益与阅读分成收益之和需达到500元。
4. 自然月内每日有效更新字数需达到4000字或6000字对应档位。
5. 保底作品上一自然月为超保底状态，下一自然月才可按规则获得全勤。
6. 同一自然月每位作家仅一部签约作品享受全勤奖励。
7. 违规、停止推荐、下架、刷收益等情况会影响福利发放。

来源：

1. [平台稿费&福利内容大揭秘](https://fanqienovel.com/writer/zone/article/7451451948726763545)
2. [2025年番茄作家福利](https://fanqienovel.com/writer/zone/article/7444811345830101017)
3. [2026年番茄小说年度作家福利](https://fanqienovel.com/writer/zone/article/7587262439981318169)

### 1.6 星火计划和年度活动

| 收益项 | 触发逻辑 | 生产影响 |
| --- | --- | --- |
| 星火计划 | 独家原创首发55分成签约非同人小说，自然月满足全勤条件，并按读完率等指标评估 | 读完率不是装饰指标，前中期章节质量直接影响奖金 |
| 全年奖/半年奖/季度奖 | 作者维度更新和完结收益等规则，按周期统计 | 要求长期稳定更新，不适合无规划断续写 |
| 优质加更奖 | 2025规则中要求追更人数等数据 | 强调追更能力和加更能力 |
| 新书追更奖 | 2025规则中要求新书字数、全勤、收益和追更指标 | 新书不能只靠开头，要在10万到80万字维持追更 |
| 拉新激励/新引力 | 根据新媒体向内容拉新、投流、广告渠道新用户等发放 | 书名、简介、开篇高光和短视频传播性会影响收益上限 |
| 短剧专项计划 | 2026年度短剧版权激励 | 改编潜力可作为方向筛选加分项 |
| 完本续签奖 | 完结后续签新书，按条件解锁奖励 | 不只看一本书收入，还看作者连续生产能力 |

重要风险：

番茄2026福利明确提到，异常发文行为包括短期内高频创建作品、高频发布章节或发布超长篇幅内容，平台可终止作品活动参与资格。

这意味着 AI 辅助不能走“高频批量灌稿”路线，只能走“高质量生产提效”路线。

### 1.7 番茄收益结论

番茄的钱主要来自：

```text
广告听读分成
+ 福利激励
+ 新媒体拉新
+ 集团渠道
+ 版权/IP
```

对新人最现实的顺序：

```text
先过签约
-> 进推荐验证
-> 争取读完率/追读率
-> 拿到基础听读收益
-> 再谈全勤/星火/拉新/版权
```

## 2. 七猫

### 2.1 签约和收益模式

可信度：high。

七猫官方模式：

1. 保底+超保底分成。
2. 纯分成。
3. 超级纯分成。

保底签约作品千字价格官方口径为20元到1000元不等。七猫官方解释保底是预付款，作品成长期给基础保障，成熟期超保底后按分成共享收益。

来源：[七猫中文网&奇妙小说网2025年作家福利](https://www.qimao.com/welfare-police/)。

### 2.2 结算和发放

| 项 | 内容 |
| --- | --- |
| 稿费发放 | 按自然月统计，次月8日发放全部稿费 |
| 收款方式 | 发放至签约时提供的银行卡 |
| 保底作品连载中 | 累计分成超过累计保底后，当月分成多发分成，否则多发保底 |
| 保底作品完结后 | 先做收支平衡，累计分成超过总稿费收入后，超出部分按分成结算 |

### 2.3 版权和打赏

| 收益项 | 官方口径 |
| --- | --- |
| 全版权收益 | 精品/知名作者保底+超保底分成模式下，全版权50%收益分成 |
| 现金打赏 | 按现金价值与作者五五分成 |
| 苹果端打赏 | 扣除30%手续费后五五分成 |
| 纵横捧场道具 | 七猫/奇妙签约作品在纵横小说APP产生的捧场收益，作者分70% |

### 2.4 全勤奖

七猫全勤比番茄更像“按合同类型和数据指标分层”的收益系统。

| 类型 | 关键条件 | 奖励逻辑 |
| --- | --- | --- |
| 保底全勤奖 | 保底独家签约作品，无断更、灌水、抄袭等不良记录 | 日更4000-6000字为5元/千字，日更6000字以上为7元/千字 |
| 纯分成阶梯全勤奖 | 纯分成独家签约作品，日更4000字以上，均留/UV等达标 | 根据平均留存、日阅读UV、类型和阶段，6元/千字到200元/千字不等 |
| 超级纯分成全勤奖 | 超级纯分成独家签约作品，日更4000字以上，按自平台千字收入分档 | 从10元/千字到1000元/千字不等 |

限制：

1. 每月最高统计20万字全勤奖。
2. 每月允许1天请假。
3. 同一作者同一月连载作品超过一本，按全勤高的作品发放。
4. 七猫和奇妙两个站点仅可享受一本作品的全勤福利。
5. 纯分成、超级纯分成签约作品若完结总字数小于20万字或被判定烂尾，完结月不发全勤。
6. 有违规行为、异常数据的作品不发全勤。

### 2.5 月票奖和扶持

七猫官方列出：

1. 纵横小说APP月票奖，月票榜第1名现金15000元，第2名10000元，第3名8000元，第4-5名5000元，第6-7名3000元，第8-10名2000元，第11-20名1000元，并配套流量资源。
2. 分成转保底和保底提价机会。
3. 征文活动高额全勤，最高千字200元。
4. 扶摇计划、爆款作品扶持、版权衍生开发、个人品牌建设等更多扶持。

### 2.6 七猫收益结论

七猫的钱主要来自：

```text
保底/超保底
+ 分成收益
+ 阶梯全勤
+ 月票/捧场
+ 征文/扶持
+ 版权衍生
```

对新人最现实的路径：

```text
先准备2万字+大纲内投
-> 争取更好的签约模式
-> 用更新和留存跑全勤/分成
-> 再争取转保底、提价和版权
```

七猫对 AI 辅助的关键风险不是“能不能写得快”，而是：

1. 官方严禁工作室投稿。
2. 同一本书多人写作会触发严重风险。
3. 灌水、抄袭、异常数据、烂尾都会影响收益。

## 3. 起点/阅文

### 3.1 官方可核收益项

可信度：medium。

当前可公开核到的完整福利说明主要来自起点中文网2020年“网络作家星计划”。这是官方来源，但时间较旧，正式投稿前必须复核当前后台和合同。

来源：[阅文集团起点中文网作家星计划](https://acts.qidian.com/2020/2020_09/index.html)。

### 3.2 作品扶持

| 收益项 | 官方旧口径 | 生产影响 |
| --- | --- | --- |
| 作家扶持 | 标准分成VIP作品，作品入V后，在起点自有平台新增单章订阅和免费收入总稿酬的20%，次月发放 | 入V和稳定VIP更新是前提 |
| 创作补贴 | 签约新书入V后，前3个月每月1500元，最多4500元 | 新人入V后有短期保障 |
| 创作激励 | 新书入V后第4个月起，满足更新和均订/收入条件，每月1000元 | 需要付费读者基础，不是无门槛全勤 |
| 文以载道 | 现实题材、传统文化及其他个性创作内容，长篇5万元起保障合约 | 更偏特定题材和编辑评估 |

关键条件：

1. 每个自然月VIP作品每日有效VIP更新字数达到4000字以上。
2. 长篇作品不能连续两个自然月单月VIP有效更新总数少于6万字。
3. 创作激励要求均订大于等于500，或当月渠道收入加全部免费收入总和大于等于2000。

### 3.3 月票和道具

起点官方旧口径包括：

1. 月票奖。
2. 道具分成。
3. 互动激励。

月票奖旧表中，起点月票榜/新书月票榜前10名有2500元到10000元不等现金奖励，并随当月稿酬发放。

### 3.4 版权/IP

起点/阅文的核心优势不是基础全勤，而是付费读者、作者品牌和IP生态。

官方作家星计划列出的品牌运营和IP服务包括：

1. 曝光资源。
2. 运营基金。
3. 版权评估报告。
4. 专属版权经纪人。
5. 版权推广策划。
6. 下游IP定制合作机会。
7. 粉丝运营和IP出圈传播。

这对新人不一定立即可达，但决定长期天花板。

### 3.5 起点收益结论

起点的钱主要来自：

```text
VIP订阅
+ 作家扶持/补贴/激励
+ 月票和道具
+ 版权/IP
+ 作者品牌运营
```

对新人最现实的路径：

```text
先过签约
-> 建立收藏/追读
-> 入V
-> 维持每日VIP更新
-> 用订阅和均订证明价值
-> 再争取福利、月票和IP机会
```

公开信息不足的地方：

1. 当前最新全勤/补贴规则。
2. 当前起点男频/女频主动申请签约门槛。
3. 当前编辑内投偏好。
4. 当前AI辅助创作政策。
5. 当前合同对多平台发表和版权授权的实际约束。

这些必须列入后续验证清单。

## 4. 三平台收益对比

| 维度 | 番茄 | 七猫 | 起点/阅文 |
| --- | --- | --- | --- |
| 主收入模型 | 免费阅读/听书广告分成 | 免费广告+保底/分成 | VIP订阅+付费生态 |
| 新人第一关 | 2万字签约机会 | 2万字+大纲投稿或直发申签 | 签约/收藏/编辑审稿 |
| 推荐/流量 | 8万字后推荐评估和7天验证 | 上架推荐+编辑/平台扶持 | 编辑、榜单、读者付费追读 |
| 稳定更新要求 | 4000/6000字、10万字后全勤门槛 | 日更4000起，月上限20万字全勤 | VIP日更4000、长篇月更约束 |
| 奖励重心 | 全勤、星火、拉新、短剧、完本续签 | 阶梯全勤、保底、月票、扶摇 | 创作补贴、创作激励、月票、IP |
| AI高频风险 | 高频发文、超长篇幅、恶意水文会影响活动资格 | 工作室、灌水、异常数据、烂尾风险高 | 付费读者和作者品牌对低质AI文容忍低 |
| 新人可达性 | 高，但竞争强且数据反馈硬 | 中，内投有编辑筛选和合同差异 | 中低到中，付费生态门槛更高 |

## 5. 对我们的直接结论

当前仍可以把番茄作为第一训练场，但理由要从“流量大”改成：

```text
番茄的直发、签约节点、推荐验证、读完率/追读率、全勤/星火/拉新/短剧激励，最适合我们先验证一个商业开局和生产运行包是否能跑。
```

但运行包不能被番茄绑死。七猫和起点分别提供两个反向校验：

1. 七猫校验“前2万字+大纲+编辑商业判断”。
2. 起点校验“付费读者追订+长期设定可信度+作者品牌”。
