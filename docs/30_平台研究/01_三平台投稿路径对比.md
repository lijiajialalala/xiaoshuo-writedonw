# 三平台投稿路径对比

## 0. 核心判断

不能只问“哪个平台赚钱”。投稿路径会决定我们前期生产哪种文本。

```text
直发验证
-> 标题、简介、前三章、前2万字、算法反馈、更新稳定性

内投/编辑投稿
-> 题材卖点、前2万字稳定性、大纲、商业预期、合同可能性

付费长线生态
-> 强开局、设定可信度、主线预期、入V后追订和读者付费意愿
```

新人现在基本都需要三章爆点或强开局。差异不是“起点要慢热、番茄要快”，而是强开局之后，平台用什么机制继续筛稿。

## 1. 番茄

### 1.1 直发签约路径

可信度：high。

官方路径：

1. 注册账号，下载番茄作家助手。
2. 完成个人信息和实名认证。
3. 创建作品并持续更新。
4. 作品达到2万字时获得第一次签约机会。
5. 作品达到5万字时获得第二次签约机会。
6. 作品达到8万字时获得第三次签约机会。
7. 三次签约评估全部失败后，该作品不再支持签约。

重要限制：

1. 安全审核不通过可以修改后重新提交。
2. 签约评估拒绝后，要到下一字数节点才能再次申请。
3. 严重抄袭、融梗等问题导致编辑评估失败，签约流程会直接终止。

官方签约评估标准包括：

1. 题材是否适合签约。
2. 是否存在不安全、低俗、言语不通、故事性不强、侵权等底线问题。
3. 是否满足基础内容规范。
4. 节奏、人物设定、开篇、代入感、主线是否满足编辑要求。

来源：[番茄签约流程优化上线通知](https://fanqienovel.com/writer/zone/article/7503776248095588377)。

### 1.2 稿费前流程和推荐路径

可信度：high。

番茄官方说明中，作品签约后才具备获得稿费的先决条件。未签约但完成实名认证的作品，主要只在番茄小说端可搜索和作者主页展示。

旧推荐入口逻辑：

1. 作品持续更新至8万字。
2. 连续更新3天。
3. 每天保证更新4000字。
4. 内容审核通过。
5. 在作品推荐模块手动点击开始推荐。

2026年推荐验证调整后：

1. 作品达到8万字并完成推荐准备后，可以主动发起推荐评估。
2. 推荐评估结果一般1-3天反馈。
3. 8万到15万字期间可随时重新发起推荐评估。
4. 15万字仍未进入推荐验证的作品，平台会自动发起推荐评估。
5. 每本作品最多经历3次推荐评估。
6. 推荐验证期为7天小流量验证。
7. 7天验证结束且字数满10万字后进入正常推荐。
8. 推荐验证期内要重点看章节读完率、章节追读率等数据。

来源：

1. [获得稿费前，你会经历哪些环节？](https://fanqienovel.com/writer/zone/article/7399504381202137150)
2. [长篇网文作品推荐验证流程调整通知](https://fanqienovel.com/writer/zone/article/7602553801446260761)

### 1.3 对生产文本的要求

番茄直发不是只拼前三章。

生产上至少要准备三层文本：

| 文本层 | 目标 |
| --- | --- |
| 书名/简介/前300字 | 让读者和编辑迅速知道主角、背景、核心钩子 |
| 前2万字 | 通过签约评估，证明有故事性、主线、主角推动力 |
| 8万-10万字 | 进入推荐评估和7天验证，证明读完率、追读率和持续更新能力 |

因此我们拆样本时不能只拆“第一章爽不爽”，还要拆：

1. 前2万字如何建立主线。
2. 8万字前是否有持续事件链。
3. 推荐验证期前是否已经有稳定读者承诺和章节追读。

## 2. 七猫

### 2.1 编辑投稿和直发投稿路径

可信度：high。

七猫官方同时提供编辑投稿和直接发文投稿。

| 投稿方式 | 材料 | 审稿时间 | 备注 |
| --- | --- | --- | --- |
| 编辑QQ邮箱投稿 | 开篇2万字正文+大纲+投稿申请表 | 1-3个工作日，节假日顺延 | 官方推荐 |
| 编辑QQ投稿 | 开篇2万字正文+大纲+投稿申请表 | 1-3个工作日，节假日顺延 | 不要一次性添加多个QQ |
| 直接注册发文投稿 | 发文满2万字后申请签约评估 | 3-7个工作日，节假日顺延 | 先发后评估 |

七猫审稿流程：

```text
作者投稿
-> 编辑初审
-> 审稿委员会复审
-> 总编终审
-> 反馈投稿结果
```

关键限制：

1. 官方明确要求不要一稿多投。
2. 官方明确严禁“工作室”投稿。
3. 工作室定义为写同一本书的不止一个作者。
4. 一经发现，可能解除协议、追回稿费、要求赔偿并追究法律责任。

来源：[七猫中文网&奇妙小说网2025年作家福利](https://www.qimao.com/welfare-police/)。

### 2.2 对生产文本的要求

七猫内投更像“作品包审核”，不是单纯算法直发。

前期必须准备：

1. 开篇2万字正文。
2. 大纲。
3. 投稿申请表。
4. 清晰卖点。
5. 可持续更新能力。

这意味着七猫样本拆解不能只看章节运动，还要看：

1. 简介卖点如何压缩给编辑。
2. 前2万字是否稳定兑现卖点。
3. 大纲是否能说明这本书后续怎么赚钱。
4. 是否具备保底、分成转保底或提价的谈判空间。

## 3. 起点/阅文

### 3.1 公开可核到的投稿和签约路径

可信度：medium。

阅文作家专区当前仍是官方入口，作家助手是官方创作工具。官方首页强调“数百名专业编辑，助力你踏上写作之旅”，并提供新手专区、写作技巧、起点创作学堂等内容。

来源：[阅文作家专区](https://write.qq.com/)。

关于签约路径，当前能核到的官方旧文说明：

1. 新书审核通过后，编辑会审阅作品，达到标准会发出签约邀请。
2. 作者也可以在满足条件后主动申请签约。
3. 旧文中男频首发站点为起点中文网/创世中文网的作品，正文字数满10万字可提交申请签约。
4. 旧文中女频首发站点作品，正文字数满5万字可提交申请签约。
5. 收到签约邀请后，通过在线签约流程完成意向确认、版权确认、合同模板查看、资料填写、审核、纸质合同寄送。

来源：[这是一份移动签约指南，请查收！](https://write.qq.com/portal/content/14626181805826801?feedType=2&lcid=35546823090990148)。

注意：该签约指南发布时间为2019年，正式投稿前必须用当前作家助手后台或编辑口径复核。

### 3.2 对生产文本的要求

起点/阅文不能简单理解成“慢热长线”。新人仍然需要强开局，但强开局服务的是：

1. 编辑是否认为作品有签约价值。
2. 读者是否愿意收藏、追读。
3. 入V后是否能稳定订阅。
4. 题材、设定、人设和主线是否能支撑长期付费。

公开资料能证明起点有强作者培训和付费/IP生态，但当前福利细则的最新官方口径不如番茄、七猫透明，必须后续进入后台或找作者经验交叉验证。

## 4. 投稿路径对运行包的影响

### 4.1 不能只做番茄版文本

小说生产运行包要能输出不同投放包。

| 投放包 | 必备内容 |
| --- | --- |
| 番茄直发包 | 书名、简介、前三章、前2万字、8万字推荐验证计划 |
| 七猫内投包 | 书名、简介、前2万字、大纲、卖点说明、更新计划 |
| 起点长线包 | 书名、简介、前3章、前10章、主线预期、入V前节奏、长期设定可信度 |

### 4.2 P2 拆书要新增平台字段

后续每本样本除了原有 E1-E8，还要记录：

```yaml
platform_submission_path:
platform_monetization_vector:
platform_review_pressure:
platform_update_pressure:
platform_violation_pressure:
```

这样拆出来的不是抽象小说技巧，而是可落地的平台生产策略。
