## 4. 使用规则

1. 官方规则可以进入平台机制文档，但金额和条件仍要在正式投稿前复核。
2. 媒体报道只能作为案例线索，不能覆盖官方规则。
3. 作者反馈至少需要记录平台、时间、题材、合同/后台证据类型。
4. 单个作者经验不能归纳成平台机制，只能触发“继续找反例和对照样本”。
5. 如果来源互相冲突，保留冲突，不强行合并。
