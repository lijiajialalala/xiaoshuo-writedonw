## 当前结论

- 顶层流水线可以沿用：
  - 精修
  - 抽取
  - source-open
  - generation / renderer
  - clean-room
  - blind review
- 小说不能只按一个颗粒度学，至少要分：
  - 书级
  - 卷 / 单元级
  - 章节级
  - 场景级
- 小说承诺不是单选，而是：
  - 主承诺
  - 副承诺
  - 不同颗粒度上的兑现方式
- 当前赛道如果偏番茄都市异常 / 高武 / 神秘事件，推进更应偏：
  - 规则识别
  - 破绽寻找
  - 局势加压
  - 信息差
  - 翻牌破局
