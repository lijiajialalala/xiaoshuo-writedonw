## 记录规则

1. 样本表记录结构观察，不复制大段正文。
2. 每条判断尽量对应具体章节、榜单位置、书名简介或平台标签。
3. 不把单本书的个例直接当成平台规律，至少横向对比后再归纳。
4. 对外部网页、截图、榜单时间要记录日期，避免后续平台变化导致误判。
5. 每条进入提取/归纳的证据都必须标注 `transfer_boundary` 和 `risk_note`。
