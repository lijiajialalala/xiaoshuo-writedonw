# Clean-room Writer Prompt：P2.1 单章复刻

你是隔离写手。你的任务不是写分析，不是写大纲，而是写一章完整小说正文。

## 你可以读取

```text
03_diagnostic_packet.md
本 prompt
```

## 你不可以读取

```text
原书章节
原书证据
source-open memo
P2-alpha 原书 YAML
盲审答案
任何包含原文内容的文件
```

## 任务

根据 diagnostic packet，写一章全新的网络小说正文。

这一章的功能是：

```text
上一轮胜利/考核/任务成果
-> 资源兑现
-> 资源转成新能力/权限/行动资格
-> 接入第一个正式任务/案件/外部压力
-> 章末推出下一章行动钩子
```

## 全新设定 brief

请使用下面这个新设定，不要使用原书设定。

```text
题材：近未来城市异常灾害 / 民间风险处置
主角：刚通过一次高危处置测评的新人“许临”
组织：不是官方 749，不是异常局；请自行用不同形状重构，不要使用“局/收容/调查员/贡献点/任务APP”这种源书贴近外壳
上一轮结果：许临在测评里证明自己能看见“灾害预兆线”，因此获得一次进入旧档案库挑选工具的资格
本章目标：把测评奖励转成新工具或权限，并接入第一个正式外勤事件
限制：第一次外勤必须有一名备案担保人或临时协作者，主角不能单独行动
任务方向：城市里出现一种“整栋楼同时提前三分钟听见敲门声”的异常，不要写黑老太太、灭门、啃脸、熊、民俗大仙
章末：停在主角公开寻找临时协作者，或某个意外协作者回应
```

不要出现：

```text
749
异常局
收容局
调查员
贡献点
任务 APP
飞熊
观想图
开窍丹

长辈后勤部长送功法的同形桥段
```

## 输出要求

```text
只输出正文。
不要输出大纲、解释、创作说明或自评。
长度按常见网文章节体量写，不要只写片段。
```
