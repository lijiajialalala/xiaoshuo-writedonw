## 1. 前一章交给第9章的压力

第8章完成了考核闭环：

```text
主角在考核中断崖式领先。
组织管理层确认他是超规格新人。
主角拿到最高奖励。
后勤部长对他产生资源倾斜。
```

第8章交给第9章的压力不是“主角还要打谁”，而是：

```text
考核奖励和组织好感必须立刻转成实质资源。
主角必须把资源变成可用战力。
战力变强后必须马上进入第一个正式任务，否则组织线会停成福利说明。
```

## 2. 第9章内部运动

第9章不是单一场景，而是四段连续运动：

```text
后勤选功法
-> 获得高价值功法/观想资源
-> 闭关突破并形成新战斗方式
-> 打开任务 APP，选择第一个正式异常案件并招募辅调
```

它的关键是：每段都把上一段的结果转成下一段压力。

```text
考核完美 -> 有资格被后勤部长照拂。
部长照拂 -> 获得比普通功法更高价值的观想资源。
观想资源 + 丹药 -> 两天突破和新攻击性能力。
战力升级 -> 主角有资格接高价值正式任务。
正式任务规则 -> 必须两人以上，逼出辅调招募。
招募喊话 -> 下一章进入搭档选择和案件调查。
```

## 3. 章节身体观察

### 3.1 段落呼吸

段落整体偏短，常用一两句推进一个动作或判断。设定说明也被拆成短段，不做长百科段。

本章在功法、境界、任务、案件说明处有信息密度，但每次说明后都会接回主角判断或行动。

### 3.2 叙述距离

叙述距离贴近主角，但不是纯第一人称内心。常见方式是：

```text
外部动作
-> 主角即时判断
-> 用口语化旁白解释为什么这件事对他有利
-> 立刻进入下一个动作
```

### 3.3 对白 / 动作 / 心理 / 设定比例

大致功能分配：

```text
前段靠邓部长对话推动资源选择。
中段靠观想画面和突破动作推动战力变化。
后段靠任务 APP 信息和主角判断推动案件入口。
章末靠论坛反应制造公开认知和下一章钩子。
```

心理不是抒情，而是利益判断、资源判断和“这个东西对我有什么用”。

## 4. 信息释放

本章释放的信息分四类：

```text
功法价值：部长推荐的不是普通货，而是高价值资源。
能力变化：主角从奖励中形成新的攻击性功法。
任务系统：正式任务有奖励、境界建议、领队/辅调分工。
案件承诺：灭门案、怪物痕迹、民俗身份、任务奖励。
```

隐藏或未完全兑现的信息：

```text
新能力的实战效果还未真正验证。
第一个案件是否只是打怪，还是有调查/误判/反咬。
辅调是谁、是否可靠。
```

章末不解释案件真相，而是停在公开招募和围观反应，把“谁来搭档、怎么出发”推给第10章。

## 5. 阻力与规则

本章没有强敌正面阻力，主要是制度和资源规则：

```text
功法选择复杂，需要更高层资源指路。
资源必须被消化成战力，不然考核奖励没有意义。
正式任务要求两人以上，主角不能单人出发。
任务信息有案件疑点，不能只靠蛮力。
```

阻力强度不高，但它完成了从组织日常到案件单元的过桥。

## 7. 第9章交给第10章的压力

第9章没有直接进入案发现场，而是停在任务招募。

它交给第10章的压力：

```text
谁会成为辅调？
主角高姿态招募会不会引发反弹？
第一个案件需要什么调查路线？
刚获得的新能力能不能在案件中发挥作用？
```

第10章承接方式是：

```text
论坛围观先兑现主角名声。
辅调申请潮兑现主角公开吸引力。
选出功能型辅调，补交通和案件分析。
通过辅调的资料分析，把第一个任务从“打怪”转成“案件调查”。
```

## 8. 可迁移机制

```text
考核/上一轮胜利后，必须立刻兑现为资源，而不是口头夸奖。
资源兑现后，必须马上转成能力、权限或下一步行动。
能力升级后，必须立刻进入新场景验证。
组织规则可以制造轻阻力，例如必须组队、申请权限、任务分工。
章末可以落在公开平台/组织频道反应上，让身份变化变成下一章压力。
```

## 10. 不确定

第9章本身不是完整案件章，只是任务入口章。它适合测试“资源兑现 -> 任务入口”单章复刻，不适合单独证明案件阻力、调查翻牌或怪物规则是否成立。
